### **졸업 카테고리**


번호 순으로 진행, 각 번호들에 대한 답변은 요약 gpt로 들어가 최종 답변 할 것임.


1. 학점
- 표 이해 gpt -> 학점 gpt -> 학점관련 답변

2. 교양
- 표 이해 gpt -> 교양 gpt -> 교양 (필수,균형) 관련 답변

3. 전공
- 아직 안함

4. 공학인증
- 진행 중

__
학점, 교양, 전공, 공학 답변 -> 요약 gpt -> 최종 답변
